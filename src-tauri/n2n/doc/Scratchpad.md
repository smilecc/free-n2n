# n2n's Scratchpad

## RPM Packaging

```
bash
./autogen.sh
./configure
make

cd packages/rpm
./configure
rpmbuild -bb ./n2n.spec
```


## Version Update

- change `VERSION` file to new version, e.g. `4.0.1`
- `git add VERSION`
- `git commit -m "moved to version 4.0.1"`
- `git tag -a 4.0.1 -m "moved to version 4.0.1"`
- `git push --tags`
